# Sign-Up Form

Página de cadastro simples desenvolvida como parte do currículo do [The Odin Project](https://www.theodinproject.com/).

## Funcionalidades

- Formulário de cadastro com campos para nome, sobrenome, email, telefone e senha
- Validação visual de campos (borda azul ao focar, borda vermelha para campos inválidos)
- Design responsivo com sidebar decorativa

## Tecnologias Utilizadas

- HTML5
- CSS3

## Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador

## Preview

O layout conta com uma imagem de fundo na sidebar e o logo "Odin" com a fonte personalizada Norse.
